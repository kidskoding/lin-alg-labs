### Setup
#### Install Numpy
1. Create a Python venv (virtual environment): `$ python3 -m venv .venv`
2. Activate the Python venv
   1. Mac and Linux: `$ .venv/bin/activate`
   2. Windows: `$ .venv\Scripts\activate`
3. Install numpy: `$ pip install numpy`
